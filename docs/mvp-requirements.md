# Image Background Remover MVP 需求文档

## 1. 项目概述

- **项目名称**：待定（建议：去背景 / CutOut / 一键抠图）
- **核心价值**：让用户最快速去除图片背景，无需注册即可使用
- **验证目标**：验证市场需求、验证变现路径（付费 API 调用 vs 免费+广告）

---

## 2. 用户故事

**作为一个普通用户：**
- 我想上传一张图片，立刻获得透明背景的版本
- 我不想注册账号、不想看教程，拖上去就用
- 我想免费下载结果到本地

**作为访客：**
- 打开页面 3 秒内知道怎么用
- 手机和电脑都能用

---

## 3. MVP 功能范围

### 必须有（MVP）
| 功能 | 描述 |
|------|------|
| 图片上传 | 支持拖拽 + 点击上传，限制jpg/png，最大 10MB |
| 背景移除 | 调用 Clipdrop API，返回透明背景图 |
| 预览 & 下载 | 实时预览结果，一键下载 PNG |
| 移动端适配 | 手机浏览友好，竖版布局 |

### 暂时不做（后续迭代）
- 用户注册/登录
- 历史记录
- 批量处理
- API Key 管理后台
- 付费订阅系统
- 广告变现

---

## 4. 用户流程

```
用户打开页面
    ↓
拖拽/选择图片（前端预览原图）
    ↓
点击"移除背景"按钮
    ↓
加载状态（骨架屏/转圈）
    ↓
展示结果（原图 vs 效果图 切换）
    ↓
点击"下载"保存 PNG
    ↓
可继续处理下一张（清空重来）
```

---

## 5. 技术方案

### 前端
- **框架**：Next.js（App Router）
- **样式**：Tailwind CSS
- **部署**：Cloudflare Pages（配合域名）
- **状态管理**：React useState（简单够用）

### 后端
- **API 路由**：Next.js API Routes（`/api/remove-bg`）
- **图片处理**：直接转发到 remove.bg，不做服务端存储
- **环境变量**：`REMOVE_BG_API_KEY` 存 Cloudflare Pages 环境变量

### 第三方服务
| 服务 | 用途 | 成本 |
|------|------|------|
| remove.bg API | 背景移除 | 免费 50次/月，付费 $0.45/100张 |

## 7. 交互细节

### 上传区域
- 默认：虚线框 + 上传图标 + "拖拽图片或点击上传"
- 拖入时：高亮边框，文字变"松开以上传"
- 上传后：显示缩略图预览，右上角有删除按钮

### 处理按钮
- 未上传时：置灰不可点
- 处理中：显示 spinner + "处理中..."
- 完成：保持可点（可重新上传新图）

### 结果展示
- 默认显示处理后图片（透明背景用棋盘格/灰白格子打底）
- 点击可切换"原图 / 结果"对比

### 下载
- 点击下载按钮，直接触发浏览器下载 PNG 文件

### 错误处理
- 文件过大：提示"图片需小于 10MB"
- API 失败：提示"处理失败，请重试"
- 非图片文件：提示"请上传 JPG/PNG 图片"
